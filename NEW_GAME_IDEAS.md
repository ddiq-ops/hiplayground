# 새로운 웹 미니게임 아이디어 제안서

## 구조 분석 요약

### 폴더 구조
```
games/[gameId]/
  ├── manifest.json  (id, title, category, difficulty, icon 등)
  ├── game.js        (Game.init(), update(), draw(), loop() 패턴)
  └── game.css       (Flexbox 기반 반응형: wrapper, header, game-area)
```

### game.js 패턴
- `Game.init(container)`: 초기화 및 레이아웃 렌더링
- `Game.update()`: 게임 로직 업데이트
- `Game.draw()`: Canvas 렌더링 (필요시)
- `Game.loop()`: requestAnimationFrame 기반 게임 루프
- 상태 관리: `Game.state` 객체
- UI 업데이트: `updateUI()` 함수

### game.css 패턴
- `.{prefix}-wrapper`: 전체 래퍼 (100% width/height, flex)
- `.{prefix}-header`: 상단 헤더 (통계/정보)
- `.{prefix}-game-area`: 게임 영역 (canvas 포함)
- `.{prefix}-controls`: 하단 컨트롤 (버튼/입력)
- 반응형: flex, aspect-ratio, max-width/height 활용

### 리소스
- 이미지 없으면 이모지(emoji) 사용
- 필요시 `assets/` 폴더 참조

## 상세 설명

### 1. block-tetris (블록 테트리스)
- **핵심 메커니즘**: 블록이 떨어지고, 회전/이동으로 줄을 완성하면 제거
- **재미 포인트**: 연쇄 제거, 빠른 판단력, 시간 압박
- **구현 요소**: 7가지 테트로미노, 회전 시스템, 레벨별 속도 증가

### 2. zombie-runner (좀비 러너)
- **핵심 메커니즘**: 좌우 이동 + 점프로 장애물과 좀비 회피, 무한 스크롤
- **재미 포인트**: 반사신경 테스트, 파워업 수집, 최고 거리 도전
- **구현 요소**: 절반 이동, 점프 물리, 파워업 효과

### 3. bubble-shooter (버블 슈터)
- **핵심 메커니즘**: 각도 조절 후 발사, 3개 이상 같은 색 맞추면 터짐
- **재미 포인트**: 전략적 각도, 연쇄 폭발, 제한된 탄약 관리
- **구현 요소**: 물리 궤적 계산, 매칭 로직, 스테이지 클리어

### 4. idle-factory (아이들 팩토리)
- **핵심 메커니즘**: 자동 생산되는 제품 판매로 돈 획득, 업그레이드로 성장
- **재미 포인트**: 수동 클릭 + 자동화, 효율적 업그레이드 선택, 점진적 성장감
- **구현 요소**: 업그레이드 트리, 자동 생산, 저장/로드

### 5. snake-classic (스네이크 클래식)
- **핵심 메커니즘**: 방향키로 이동, 아이템 먹으면 길어짐, 벽/몸 충돌 시 게임오버
- **재미 포인트**: 단순하지만 중독성, 길어질수록 난이도 상승, 최고 점수 도전
- **구현 요소**: 그리드 기반 이동, 충돌 감지, 점수 시스템

### 6. match-3-puzzle (매치 3 퍼즐)
- **핵심 메커니즘**: 인접 블록 교환으로 3개 이상 매칭, 중력으로 블록 떨어짐
- **재미 포인트**: 빠른 스와이프, 특수 블록 조합, 목표 달성 전략
- **구현 요소**: 매칭 알고리즘, 특수 블록 생성, 목표 시스템

### 7. pong-battle (퐁 배틀)
- **핵심 메커니즘**: 상하 이동 패들로 공을 튕김, 상대편 골대에 넣으면 점수
- **재미 포인트**: 빠른 반응, 각도 조절, AI와의 대전
- **구현 요소**: 물리 충돌, AI 패턴, 점수 시스템

### 8. 2048-merge (2048 머지)
- **핵심 메커니즘**: 상하좌우 이동으로 같은 숫자 블록 합치기, 2048 달성 목표
- **재미 포인트**: 전략적 계획, 큰 수 만들기의 성취감, 게임 오버 예측
- **구현 요소**: 그리드 이동, 합치기 로직, 새로운 블록 생성

### 9. plane-shooter (플레인 슈터)
- **핵심 메커니즘**: 좌우 이동으로 적 비행기 격추, 파워업 수집, 보스전
- **재미 포인트**: 탄막 회피, 파워업 조합, 보스 패턴 학습
- **구현 요소**: 적 스폰 시스템, 파워업 효과, 보스 AI

### 10. breakout-blocks (브레이크아웃 블록)
- **핵심 메커니즘**: 패들로 공 튕기기, 블록 맞추면 깨짐, 모든 블록 제거 목표
- **재미 포인트**: 정확한 각도 조절, 특수 블록 효과, 스테이지 클리어
- **구현 요소**: 물리 충돌, 블록 파괴, 스테이지 진행
