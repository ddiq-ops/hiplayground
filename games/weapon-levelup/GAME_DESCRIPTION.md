# 내 무기만 레벨업 게임 설명서

## 게임 개요
무기를 강화하고 판매하여 골드를 모으는 전략 게임입니다. 대장간에서 무기를 강화하고, 상점에서 판매하여 더 많은 골드를 얻어 더 높은 레벨의 무기를 만들 수 있습니다.

## 게임 규칙

### 기본 시스템
- **초기 골드**: 100 골드
- **초기 무기 레벨**: 1
- **게임 목표**: 무기를 최대한 높은 레벨까지 올리고, 판매하여 골드를 획득

### 대장간 (강화 시스템)

#### 강화 비용 계산
- 기본 비용: `레벨 × 15 + 10` 골드
- 인플레이션 적용: `기본 비용 × (1 + 판매횟수 × 0.01)`
  - 판매 1회: +1% 증가
  - 판매 2회: +2% 증가
  - 판매 10회: +10% 증가

#### 성공 확률
- 기본 확률: `100% - (레벨 × 4)%`
- 최소 확률: 10% (레벨이 높아도 최소 10%는 보장)

#### 강화 결과
- **성공 시**:
  - 무기 레벨 +1
  - 골드 보상 없음
  - 성공 애니메이션 및 "강화 성공!" 메시지 표시
  
- **실패 시**:
  - 무기 레벨이 1로 떨어짐
  - 실패 애니메이션 및 "강화 실패!" 메시지 표시
  - 게임오버 조건 체크

#### 강화 프로세스
1. "무기 강화하기" 버튼 클릭
2. 버튼이 "강화 중..."으로 변경되고 비활성화
3. 무기에 강화 애니메이션 시작 (반짝임, 회전 효과)
4. 1.5초 후 결과 표시
5. 성공/실패에 따른 애니메이션 및 메시지 표시

### 상점 (판매 시스템)

#### 판매 가격 계산
- 판매 가격: `레벨² × 60 + 50` 골드
- 레벨이 높을수록 기하급수적으로 비싸게 판매 가능

#### 판매 규칙
- 레벨 1 무기는 판매 불가
- 판매 시:
  - 골드 획득
  - 무기 레벨이 1로 리셋
  - 판매 횟수 +1 (인플레이션 증가)
  - 게임오버 상태 해제

### 게임오버 조건
- **조건**: 레벨 1 상태에서 레벨 2로 올라가기 위한 골드가 부족할 때
- 레벨 2 강화 비용: `1 × 15 + 10 = 25` 골드 (인플레이션 적용 전)
- 게임오버 시:
  - 게임오버 화면 표시
  - 최종 통계 표시 (최종 레벨, 보유 골드, 총 강화 시도, 성공 횟수)
  - "다시 시작하기" 버튼으로 게임 리셋

## 게임 통계

### 추적되는 통계
- 총 강화 시도 횟수
- 성공한 강화 횟수
- 성공률 (성공 횟수 / 총 시도 횟수)
- 판매 횟수 (인플레이션 계산용)

## UI 요소

### 메인 화면
- **골드 표시**: 현재 보유 골드
- **무기 레벨**: 현재 무기 레벨
- **성공률**: 다음 강화 성공 확률
- **인플레이션 알림**: 판매 횟수에 따른 비용 증가율 표시

### 무기 표시
- 레벨에 따라 다른 아이콘 표시:
  - 레벨 1-4: ⚔️
  - 레벨 5-9: 🗡️
  - 레벨 10-14: 🔪
  - 레벨 15-19: ⚔️
  - 레벨 20+: 🗡️

### 애니메이션
- **강화 중**: 무기 반짝임 및 회전 효과
- **강화 성공**: 무기 확대 펄스 + 녹색 글로우
- **강화 실패**: 무기 흔들림 + 빨간색 글로우
- **판매**: 무기 회전 애니메이션

## 게임 밸런스

### 난이도 요소
1. **실패 시 리스크**: 실패하면 레벨 1로 떨어짐
2. **인플레이션**: 판매할수록 강화 비용 증가
3. **성공 확률 감소**: 레벨이 높아질수록 성공 확률 감소
4. **골드 보상 없음**: 성공해도 골드를 받지 않음

### 전략 요소
- 언제 판매할지 결정 (높은 레벨에서 판매 vs 계속 강화)
- 골드 관리 (게임오버 방지)
- 리스크 관리 (실패 시 레벨 1로 떨어지는 리스크)

## 게임 팁

1. **초반 전략**: 레벨 3-5 정도에서 첫 판매를 고려
2. **골드 관리**: 항상 레벨 2 강화 비용 이상의 골드를 유지
3. **리스크 관리**: 높은 레벨에서는 판매를 고려하여 리스크 회피
4. **인플레이션 고려**: 판매 횟수가 많아질수록 강화 비용이 증가하므로 신중하게 판매 결정
